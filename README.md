# Aesthetic Clock

A beautiful, customizable clock with glass morphism design, cursor effects, and multiple display modes.

## Features

- **Multiple Display Modes**: Analog, Digital, or Both
- **Clock Faces**: Numbers, Markers, Roman, Minimal, Diamond, Dots, Geometric, Hexagon
- **Customizable**: Colors, fonts, sizes, glow effects
- **Fullscreen Mode**: Press F or click the button
- **Save Presets**: Save your favorite configurations
- **Randomize**: Generate random aesthetic combinations

## Usage

Open `aesthetic-clock.html` in any modern browser.

## Deploy to GitHub Pages

1. Push this repo to GitHub
2. Go to Settings > Pages
3. Select "main" branch and "/ (root)" folder
4. Your clock will be live at `https://yourusername.github.io/aesthetic-clock`
